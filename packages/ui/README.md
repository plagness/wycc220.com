# UI package

Общие визуальные примитивы и токены платформы. Пакет не знает о маршрутах, API и бизнес-логике.

Новый компонент добавляется сюда, если он нужен минимум двум независимым страницам или приложениям. Составной блок конкретного раздела остаётся в соответствующем feature-модуле `apps/web`.

Каждый компонент обязан иметь типизированные props, доступное HTML-представление, состояния focus/hover/disabled и не содержать сетевых запросов. Правила визуального языка находятся в [`docs/design.system.md`](../../docs/design.system.md).
