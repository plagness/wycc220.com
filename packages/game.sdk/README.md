# Game SDK

Минимальный безопасный мост между игрой и host-платформой. SDK не выдаёт игре cookies, ключи API или прямой доступ к аккаунту пользователя.

До утверждения протокола пакет содержит только версионируемый envelope. Любое новое сообщение документируется, валидируется с обеих сторон и не должно принимать произвольные команды выполнения.
