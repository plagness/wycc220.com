# Политика данных

## Принципы

Собираем минимум, объясняем назначение каждого поля, не продаём данные, не строим скрытые профили и не объединяем аккаунты без подтверждения владельца. Сырые сообщения чатов не являются публичным архивом проекта.

## Предлагаемые сроки хранения

До юридической и продуктовой проверки это консервативные ограничения, а не разрешение начать сбор:

- сырые события чата — не хранить либо удалить после дедупликации и агрегации не позднее 7 дней;
- нормализованные события активности без текста — до 90 дней для пересчёта и антифрода;
- дневные агрегаты — пока профиль участвует в рейтинге;
- audit log модерации — 1 год с ограниченным доступом;
- удалённые изображения — физическое удаление из хранилища и CDN в разумный технический срок.

Перед запуском каждого коллектора необходимо проверить условия API соответствующей платформы, основание обработки, процедуру удаления и возрастные риски.

## Права участника

Предусматриваются понятные запросы на просмотр привязанных данных, исправление, отвязку площадки, исключение из рейтинга и удаление данных, которыми проект вправе распоряжаться. Такие запросы не требуют публичного доказательства личности; подтверждение выполняется через владение соответствующим аккаунтом.
