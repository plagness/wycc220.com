# Дизайн-система

## Направление

Визуальный язык вдохновлён атмосферой игрового и стримингового контента вокруг wycc220/elwycco: тёмный фон, высокий контраст, прямолинейная крупная типографика, интерфейсные метки и один энергичный акцент. Это самостоятельная система фан-проекта, а не копия оформления каналов.

Нельзя использовать официальный аватар, логотип, фотографии или кадры как системный бренд без отдельного разрешения. В карточке конкретного видео допустима официальная thumbnail с исходной платформы в рамках её API и правил показа.

## Характер

- прямой и немного суровый, но не враждебный;
- игровой, но не перегруженный неоном;
- живой за счёт контента, а не декоративного шума;
- ироничный в микротексте, точный в ошибках и правилах;
- community-first: автор работы заметнее интерфейса платформы.

## Токены

Источник истины — `packages/ui/src/styles.css`.

| Роль            | Токен             | Значение                     |
| --------------- | ----------------- | ---------------------------- |
| Холст           | `--color-canvas`  | почти чёрный нейтральный фон |
| Поверхность     | `--color-surface` | карточки и панели            |
| Текст           | `--color-text`    | тёплый белый                 |
| Вторичный текст | `--color-muted`   | холодный серый               |
| Акцент          | `--color-accent`  | кислотный лайм               |
| Ошибка          | `--color-danger`  | красный, только для проблем  |

Акцент нельзя применять ко всему одновременно. Он отмечает действие, текущий статус или один главный смысл экрана.

## Композиция

Используется видимая сетка, тонкие границы и крупные свободные поля. Скругления не являются основным приёмом; интерфейс строится на прямых панелях. Заголовки короткие, плотные и контрастные. Основной текст сохраняет комфортную ширину и межстрочный интервал.

## Контент

Thumbnail и фанатская работа показываются без агрессивных цветовых фильтров. Всегда отображаются автор/источник, площадка и переход к оригиналу. Нельзя добавлять fake-live, искусственную срочность, autoplay со звуком и механики, скрывающие источник.

## Доступность

- контраст текста соответствует WCAG AA;
- интерфейс доступен с клавиатуры;
- цвет не является единственным носителем смысла;
- анимация учитывает `prefers-reduced-motion`;
- минимальная интерактивная область — 44×44 px;
- изображения имеют осмысленный alt либо пустой alt для декора.

Перед расширением палитры или добавлением нового визуального паттерна команда обновляет этот документ и пакет `@wycc220/ui`.
