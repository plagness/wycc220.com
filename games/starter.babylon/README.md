# Babylon.js game starter

Рабочий минимальный шаблон 3D-игры. Он использует WebGL как максимально совместимую основу для браузеров и Telegram WebView. WebGPU добавляется как progressive enhancement после обязательного тестирования fallback.

```bash
pnpm --filter @wycc220/game.starter.babylon dev
```

Перед началом игры скопируйте папку в собственный fork, измените package name и `public/game.manifest.json`. Инструкции, лимиты и публикация описаны в [`docs/game.development.md`](../../docs/game.development.md).
