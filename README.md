# wycc220.com

Открытая некоммерческая фан-платформа сообщества вокруг творчества **wycc220 / elwycco**: единый каталог видео и стримов, фанатские игры и арты, открытое API и прозрачная статистика активности.

> Проект создан фанатами и не является официальным ресурсом wycc220/elwycco, YouTube, Twitch или Telegram.

## Философия

Платформа принадлежит сообществу. Мейнтейнеры обеспечивают безопасность, качество и равные правила, но не присваивают себе авторство работ участников. Здесь нет платного доступа, сбора пожертвований, рекламы, продажи пользовательских данных и саморекламы. Единственная цель проекта — фанатское творчество и удобный открытый архив.

Основные принципы закреплены в [charter.md](./docs/charter.md), а ожидаемое поведение — в [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).

## Локальный запуск

Требуется Node.js 22+ и Corepack.

```bash
corepack enable
pnpm install --frozen-lockfile
pnpm dev
```

Скопируйте `.env.example` в `.env` только при необходимости. Реальные ключи никогда не добавляются в Git.

## Ветки

- `main` — то, что разрешено публиковать на основном сайте;
- `dev` — интеграция и тестирование будущих изменений.

Других постоянных веток нет. Короткоживущие ветки контрибьюторов существуют в их форках и вливаются pull request в `dev`. Релизный PR идёт из `dev` в `main`.
