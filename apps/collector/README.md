# Collector

Здесь будут независимые адаптеры официальных API YouTube, Twitch и Telegram. Папка намеренно пока не содержит интеграционного кода: сбор активности нельзя запускать до утверждения `docs/DATA-POLICY.md`, проверки условий каждой платформы и проектирования opt-out.

Каждый адаптер обязан:

- использовать отдельные минимально привилегированные credentials;
- не логировать токены и тексты сообщений;
- быть идемпотентным по внешнему event id;
- поддерживать удаление и остановку без потери остальных источников;
- отдавать нормализованное событие из `packages/contracts`.
