# Кодекс сообщества

Мы создаём спокойное, открытое и безопасное пространство для фанатского творчества. Уважайте авторов, участников и людей, чья публичная активность отображается на платформе.

Недопустимы травля, угрозы, публикация личных данных, дискриминация, спам, самореклама, коммерческие предложения, манипуляция рейтингом и выдача проекта за официальный ресурс блогера.

Модераторы применяют правила одинаково ко всем, объясняют решения и по возможности оставляют путь для исправления. Срочное скрытие контента допустимо при риске для людей, безопасности или авторских прав; решение затем документируется и может быть обжаловано.

Сообщить о проблеме можно приватно способом из [SECURITY.md](./SECURITY.md). Публичный issue не подходит для секретов, персональных данных и жалоб, способных навредить человеку.
